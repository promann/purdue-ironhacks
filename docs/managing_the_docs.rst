Creating the docs
-----------------

Initial Start
~~~~~~~~~~~~~

Follow the `Get
Started <http://docs.readthedocs.io/en/latest/getting_started.html>`__
page and run:

1. ``pip install sphinx sphinx-autobuild``
2. ``mkdir docs``
3. ``sphinx-quickstart``
4. ``make html``

Theme settings
~~~~~~~~~~~~~~

.. code:: sh

    pip install sphinx_rtd_theme

And then set it in the ``conf.py``.
